# UI 구조

## 전체 구성

```
익스텐션
├── 팝업 (popup.html)           # 툴바 아이콘 클릭 시
├── 리스트 페이지 (list.html)    # 저장된 회사 목록
├── 상세 페이지 (detail.html)    # 회사별 상세 정보
└── 컨펌 팝업 (모달)             # 추출 후 확인
```

## 1. 팝업 (popup.html)

익스텐션 아이콘 클릭 시 표시되는 메인 인터페이스.

```
┌─────────────────────────────┐
│  🏢 AI 기업분석              │
├─────────────────────────────┤
│  현재: wanted.co.kr ✓       │
│                             │
│  [📝 텍스트 추출]            │
│  [📊 그래프 캡처]            │
│  [📁 PDF 업로드]             │
│                             │
│  ─────────────────────────  │
│  저장된 회사: 5개            │
│  [📋 회사 목록 보기 →]       │
└─────────────────────────────┘
```

### 기능

| 요소 | 동작 |
|------|------|
| 현재 사이트 표시 | 지원 사이트면 ✓, 미지원이면 ✗ |
| 텍스트 추출 | 요소 선택 모드 활성화 |
| 그래프 캡처 | 영역 선택 모드 활성화 |
| PDF 업로드 | 파일 선택 다이얼로그 |
| 회사 목록 보기 | 리스트 페이지로 이동 (새 탭) |

### 지원하지 않는 사이트

버튼 비활성화 + "지원하지 않는 사이트입니다" 메시지 표시

## 2. 리스트 페이지 (list.html)

저장된 회사 목록을 표시하는 전체 페이지.

```
┌─────────────────────────────────────────────────┐
│  🏢 저장된 회사 목록                              │
├─────────────────────────────────────────────────┤
│                                                 │
│  정렬: [최근순 ▼]  검색: [___________🔍]         │
│                                                 │
│  ┌─────────────────────────────────────────┐   │
│  │ 카카오                      ⭐ 4.2  🗑️  │   │
│  │ 원티드 ✓  혁신의숲 ✓  블라인드 ✓         │   │
│  │ 최근 수집: 2025-01-15                    │   │
│  └─────────────────────────────────────────┘   │
│                                                 │
│  ┌─────────────────────────────────────────┐   │
│  │ 토스                            분석 전  │   │
│  │ 원티드 ✓  잡플래닛 ✓                     │   │
│  │ 최근 수집: 2025-01-14                    │   │
│  └─────────────────────────────────────────┘   │
│                                                 │
│  ┌─────────────────────────────────────────┐   │
│  │ 당근마켓                         ⭐ 3.8  │   │
│  │ 혁신의숲 ✓  DART ✓  블라인드 ✓           │   │
│  │ 최근 수집: 2025-01-10                    │   │
│  └─────────────────────────────────────────┘   │
│                                                 │
└─────────────────────────────────────────────────┘
```

### 기능

| 요소 | 동작 |
|------|------|
| 정렬 드롭다운 | 최근순 / 이름순 / 점수순 |
| 검색 | 회사명으로 필터링 |
| 회사 카드 | 클릭 시 상세 페이지로 이동 |
| 점수 표시 | AI 분석 완료 시 ⭐ 점수, 미완료 시 "분석 전" |
| 소스 배지 | 수집된 데이터 소스 표시 |
| 삭제 버튼 | 회사 삭제 (확인 다이얼로그) |

## 3. 상세 페이지 (detail.html)

회사별 상세 정보와 AI 분석 결과를 표시.

```
┌─────────────────────────────────────────────────┐
│  ← 목록    카카오                    🗑️ 삭제    │
├─────────────────────────────────────────────────┤
│                                                 │
│  ┌─ AI 분석 결과 ─────────────────────────────┐ │
│  │                                            │ │
│  │  총점: ⭐ 4.2 / 5.0                        │ │
│  │                                            │ │
│  │  📊 예상 Runway: 18개월                    │ │
│  │  ⚠️ 재무 리스크: 중간                      │ │
│  │  💬 리뷰 요약: 워라밸 좋음, 성장 기회 많음   │ │
│  │                                            │ │
│  │  [🔄 재분석]                               │ │
│  │                                            │ │
│  └────────────────────────────────────────────┘ │
│                                                 │
│  ┌─ 수집된 데이터 ────────────────────────────┐ │
│  │                                            │ │
│  │  ── 원티드 (2025-01-15) ──                 │ │
│  │  설립: 2010년 (14년차)                     │ │
│  │  매출: 1조 2000억                          │ │
│  │  평균연봉: 8,500만원                       │ │
│  │  사원수: 3,200명                           │ │
│  │  [📊 매출 그래프 보기]                     │ │
│  │                                            │ │
│  │  ── 혁신의숲 (2025-01-14) ──               │ │
│  │  현재 인원: 3,150명                        │ │
│  │  연간 퇴사율: 12%                          │ │
│  │  [📊 입퇴사 그래프 보기]                   │ │
│  │  ... (3개년 재무 테이블)                   │ │
│  │                                            │ │
│  │  ── 블라인드 (2025-01-13) ──               │ │
│  │  총평: 4.1점                               │ │
│  │  리뷰 3개                                  │ │
│  │                                            │ │
│  └────────────────────────────────────────────┘ │
│                                                 │
│  [+ 데이터 추가하기]                             │
│                                                 │
└─────────────────────────────────────────────────┘
```

### 기능

| 요소 | 동작 |
|------|------|
| 목록 버튼 | 리스트 페이지로 돌아가기 |
| 삭제 버튼 | 회사 삭제 (확인 다이얼로그) |
| AI 분석 결과 | 점수, Runway, 리스크, 리뷰 요약 표시 |
| 재분석 버튼 | AI 분석 다시 실행 |
| 수집된 데이터 | 소스별 데이터 스크롤 표시 |
| 그래프 보기 | 저장된 그래프 이미지 모달로 표시 |
| 데이터 추가하기 | 수집 가이드 모달 표시 |

### 분석 전 상태

```
┌─ AI 분석 결과 ─────────────────────────────┐
│                                            │
│  분석 전                                    │
│                                            │
│  재무 데이터: ✓                             │
│  리뷰 데이터: ✗ (필요)                      │
│                                            │
│  [분석하기] (비활성화)                       │
│                                            │
└────────────────────────────────────────────┘
```

## 4. 데이터 추가 안내 모달

"데이터 추가하기" 클릭 시 표시.

```
┌─────────────────────────────────────────────────┐
│  📖 데이터 수집 가이드                           │
├─────────────────────────────────────────────────┤
│                                                 │
│  📌 기업 정보                                    │
│  → 원티드에서 회사 검색 → 회사 상세 페이지        │
│     https://wanted.co.kr                        │
│                                                 │
│  📌 재무 정보 (일반 기업)                        │
│  → DART에서 회사 검색 → 사업보고서 → PDF 다운    │
│     https://dart.fss.or.kr                      │
│                                                 │
│  📌 재무 정보 (스타트업)                         │
│  → 혁신의숲에서 검색 → 조직/재무 탭              │
│     https://innoforest.co.kr                    │
│  → 중기벤처확인에서 검색 → 재무정보 탭           │
│     https://smes.go.kr/venturein                │
│                                                 │
│  📌 리뷰                                        │
│  → 블라인드에서 회사 검색 → 리뷰 페이지          │
│     https://blind.com                           │
│  → 잡플래닛에서 회사 검색 → 리뷰 페이지          │
│     https://jobplanet.co.kr                     │
│                                                 │
│                              [닫기]             │
└─────────────────────────────────────────────────┘
```
