# Wanted Helper

원티드(Wanted) 사이트를 더 편리하게 사용할 수 있도록 도와주는 크롬 확장 프로그램입니다.

> **⚠️ 면책조항 (Disclaimer)**
> - 본 프로젝트는 개인 포트폴리오 및 학습 목적으로 제작되었습니다.
> - "원티드(Wanted)"는 원티드랩 주식회사의 등록 상표입니다.
> - 본 프로젝트는 원티드랩 주식회사와 공식적인 관계가 없으며, 후원 또는 보증을 받지 않습니다.
> - 상업적 목적으로 사용되지 않습니다.

## 주요 기능

### 1. 관심 없는 회사/채용 공고 관리
- 관심 없는 회사를 저장하면 해당 회사의 모든 채용 공고가 흐리게 표시됩니다
- 관심 없는 채용 공고를 개별적으로 저장할 수 있습니다
- 저장한 정보는 크롬 계정에 동기화되어 다른 기기에서도 사용 가능합니다

### 2. 지원 완료 자동 관리
- 채용 공고에 지원하면 자동으로 해당 공고가 관심 없음으로 표시됩니다
- 중복 지원을 방지하고 목록을 깔끔하게 관리할 수 있습니다

### 3. 실시간 필터링
- 채용 공고 목록 페이지에서 관심 없는 공고가 자동으로 흐리게 표시됩니다
- 페이지 이동 시에도 자동으로 적용됩니다

## 설치 방법

### 1단계: 파일 다운로드 및 압축 해제
받은 `wanted_helper.zip` 파일을 적당한 위치에 압축 해제합니다.

> ⚠️ **중요:** 압축 해제한 폴더는 삭제하지 마세요! 확장 프로그램이 이 폴더를 계속 참조합니다.

### 2단계: Chrome 확장 프로그램 페이지 열기
Chrome 브라우저에서 다음 중 한 가지 방법으로 확장 프로그램 관리 페이지를 엽니다:
- 주소창에 `chrome://extensions/` 입력
- 또는 메뉴 > 도구 더보기 > 확장 프로그램

### 3단계: 개발자 모드 활성화
우측 상단의 **"개발자 모드"** 토글을 켭니다.

### 4단계: 확장 프로그램 로드
1. **"압축해제된 확장 프로그램을 로드합니다"** 버튼을 클릭합니다
2. 1단계에서 압축 해제한 폴더를 선택합니다
3. "My Wanted Helper" 확장 프로그램이 목록에 표시되면 설치 완료!

## 사용 방법

### 회사 페이지에서
1. 원티드에서 관심 없는 회사 페이지로 이동합니다 (예: `https://www.wanted.co.kr/company/12345`)
2. 우측 사이드바에 **"관심 없음 추가"** 버튼이 표시됩니다
3. 버튼을 클릭하면 해당 회사가 관심 없음 목록에 추가됩니다
4. 다시 클릭하면 목록에서 제거됩니다

### 채용 공고 상세 페이지에서
1. 채용 공고 상세 페이지로 이동합니다 (예: `https://www.wanted.co.kr/wd/12345`)
2. 우측 사이드바에 **"관심 없음 추가"** 버튼이 표시됩니다
3. 버튼을 클릭하면 해당 채용 공고만 관심 없음 목록에 추가됩니다
4. 이미 지원한 공고는 자동으로 관심 없음 목록에 추가됩니다

### 채용 공고 목록 페이지에서
- 원티드 메인이나 검색 결과 페이지에서 관심 없는 회사/공고가 자동으로 흐리게(투명도 50%) 표시됩니다
- 별도의 조작 없이 자동으로 적용됩니다

## 문제 해결

### 확장 프로그램이 작동하지 않아요
1. Chrome 확장 프로그램 페이지에서 확장 프로그램이 **활성화** 상태인지 확인하세요
2. 페이지를 새로고침(F5)해보세요
3. 버튼이 보이지 않는다면 페이지 구조가 변경되었을 수 있습니다

### 버튼이 보이지 않아요
- 원티드 사이트의 회사 페이지나 채용 공고 상세 페이지에서만 버튼이 표시됩니다
- 목록 페이지에서는 버튼이 표시되지 않고 필터링만 적용됩니다

### 저장한 정보를 초기화하고 싶어요
1. Chrome 확장 프로그램 페이지에서 "My Wanted Helper"를 찾습니다
2. 세부정보 클릭 > 사이트 데이터 지우기
3. 또는 확장 프로그램을 제거 후 재설치하세요

## 업데이트 방법

새 버전을 받으셨다면:
1. Chrome 확장 프로그램 페이지에서 기존 "My Wanted Helper"를 **제거**합니다
2. 새로운 파일로 위의 설치 방법을 다시 따라하세요
3. 저장된 관심 없음 데이터는 Chrome 동기화에 저장되어 있어 유지됩니다

## 기술 정보

- **대상 사이트:** https://www.wanted.co.kr
- **저장 방식:** Chrome Storage Sync API
- **권한:**
  - `storage`: 관심 없음 데이터 저장
  - `activeTab`: 현재 탭 접근
  - `scripting`: 페이지 스크립트 실행
  - `webRequest`: 웹 요청 모니터링
